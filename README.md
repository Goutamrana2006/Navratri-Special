# 🌸 Happy Navratri Greeting Web App
An interactive and visually appealing web project created to celebrate the vibrant festival of **Navratri**.
This project showcases creativity with animations, slideshow, and festive effects.

# 🎯 Features
✨ **Festive Animated UI**
* Dynamic 9-color glowing background 🎨
* Smooth fade-in card animation
* Beautiful Durga Maa background

🔥 **Fireworks Animation**
* Random colorful fireworks on screen
* Enhances festive experience 🎆

🪔 **9 Days Goddess Slideshow**
* Displays all **Navratri Goddesses (Navdurga)**
* Auto-changing images with names

🎵 **Background Music**
* Auto-play festive music (with user interaction support)

📱 **Responsive Design**
* Works on mobile and desktop

# 🛠️ Tech Stack

* HTML5
* CSS3 (Animations & Effects)
* JavaScript (DOM, Events, Animations)
